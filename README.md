# MultiPost Desktop Release

This is the release repository for [MultiPost Desktop](https://github.com/leaperone/MultiPost-Desktop) - a multi-platform content publishing desktop application.

## Download

Visit the [Releases](https://github.com/leaperone/MultiPost-Desktop-Release/releases) page to download the latest version.

### Available Platforms

| Platform | Architecture | File Type |
|----------|-------------|-----------|
| macOS | Intel (x64) | `.dmg` |
| macOS | Apple Silicon (arm64) | `.dmg` |
| Windows | x64 | `.exe` |
| Linux | x64 | `.AppImage`, `.deb`, `.snap` |

## Installation

### macOS

1. Download the `.dmg` file for your architecture
2. Open the `.dmg` file
3. Drag MultiPost to your Applications folder

### Windows

1. Download the `.exe` installer
2. Run the installer
3. Follow the installation wizard

### Linux

**AppImage:**
```bash
chmod +x MultiPost-*.AppImage
./MultiPost-*.AppImage
```

**Debian/Ubuntu:**
```bash
sudo dpkg -i multipost-desktop_*.deb
```

**Snap:**
```bash
sudo snap install multipost-desktop_*.snap --dangerous
```

## Features

- Multi-platform social media publishing
- Support for 50+ platforms including:
  - Weibo, Bilibili, Xiaohongshu, Douyin
  - Twitter, Facebook, LinkedIn
  - And many more...
- Local-first architecture
- Privacy-focused design
